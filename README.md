# GPU Resource Allocation Simulation

This Python script simulates a simplified GPU allocation scenario. It demonstrates how a basic, unmanaged allocation strategy can lead to unallocated jobs even when total GPU capacity might be sufficient, illustrating the core problem of GPU allocation governance.

## Language

`python`

## How to Run

Run the script using: python gpu_allocation.py
